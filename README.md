# tmux_enviroment_deployer
Auto preconfigured Tmux deployer for Typst developments. 

## Required packages
- tmux
- zathura
- nvim
- typst

## SYNOPSIS
    ./tmux_enviroment_deployer [project_path]

## REQUIRED FOLDER STRUCTURE
    GLOBAL_PROJECT_FOLDER_PATH/
        |-pdf/
            |-GLOBAL_PROJECT_FOLDER_PATH.typ
            |-GLOBAL_PROJECT_FOLDER_PATH.pdf
            |-plantilla.typ
